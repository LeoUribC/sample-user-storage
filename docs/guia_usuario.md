# Guía de Usuario

Esta guía explica cómo utilizar la aplicación de línea de comandos para gestionar usuarios.

## 🚀 Ejecutar la aplicación

La aplicación se ejecuta utilizando Python:

```bash
uv run main.py [COMANDO] [INPUTS]
```

!!! important "Comandos específicos"
    Los comandos específicos varían según la aplicación, esta es una plantilla de referencia.
